# Node Demo

<img src="node-xrp-demo.png" alt="Screenshot of the Xpring SDK XRP Demo"/>

Demonstrates the XRP API in [Xpring-JS](http://github.com/xpring-eng/xpring-js). 

To build:
```shell
# Install dependencies
npm i

# Run
node src/index-xrp.js
```
